# 响应式布局

本文档描述 RBAC 系统的响应式布局适配规则。

---

## 断点定义

| 断点名称 | 范围 | 设备类型 |
|---------|------|---------|
| **xs** | < 640px | 小手机 |
| **sm** | 640px - 768px | 大手机 |
| **md** | 768px - 1024px | 平板 |
| **lg** | 1024px - 1280px | 小桌面 |
| **xl** | 1280px - 1536px | 大桌面 |
| **2xl** | > 1536px | 超大桌面 |

### CSS Media Query

```css
/* xs */
@media (max-width: 639px) {}

/* sm */
@media (min-width: 640px) and (max-width: 767px) {}

/* md */
@media (min-width: 768px) and (max-width: 1023px) {}

/* lg */
@media (min-width: 1024px) and (max-width: 1279px) {}

/* xl */
@media (min-width: 1280px) and (max-width: 1535px) {}

/* 2xl */
@media (min-width: 1536px) {}
```

## Vue 响应式工具

### 响应式状态 Hook

```javascript
// hooks/useResponsive.js
import { ref, onMounted, onUnmounted } from 'vue'

export function useResponsive() {
  const breakpoint = ref('lg')

  const updateBreakpoint = () => {
    const width = window.innerWidth
    if (width < 640) breakpoint.value = 'xs'
    else if (width < 768) breakpoint.value = 'sm'
    else if (width < 1024) breakpoint.value = 'md'
    else if (width < 1280) breakpoint.value = 'lg'
    else if (width < 1536) breakpoint.value = 'xl'
    else breakpoint.value = '2xl'
  }

  onMounted(() => {
    updateBreakpoint()
    window.addEventListener('resize', updateBreakpoint)
  })

  onUnmounted(() => {
    window.removeEventListener('resize', updateBreakpoint)
  })

  return {
    breakpoint,
    isMobile: computed(() => breakpoint.value === 'xs' || breakpoint.value === 'sm'),
    isTablet: computed(() => breakpoint.value === 'md'),
    isDesktop: computed(() => breakpoint.value === 'lg' || breakpoint.value === 'xl' || breakpoint.value === '2xl')
  }
}
```

### 使用示例

```vue
<template>
  <div class="page">
    <!-- 移动端显示卡片列表 -->
    <template v-if="isMobile">
      <UserCard v-for="user in users" :key="user.id" :user="user" />
    </template>

    <!-- 桌面端显示表格 -->
    <template v-else>
      <DataTable :columns="columns" :data="users" />
    </template>
  </div>
</template>

<script setup>
import { useResponsive } from '@/hooks/useResponsive'

const { isMobile, isDesktop } = useResponsive()
</script>
```
